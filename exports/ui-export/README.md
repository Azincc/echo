# Echoes UI Export

生成时间: 2026-03-30

这个导出包整理了项目当前仓库中已有的界面截图，并附带一个可直接打开的本地画廊页:

- `index.html`: 本地可浏览的 UI 画廊
- `manifest.json`: 截图与页面来源清单
- `screens/`: 导出的截图文件

建议使用方式:

1. 直接在浏览器打开 `index.html`
2. 或发送 `ui-export.zip` 给其他人预览

说明:

- 本导出基于仓库中的现有截图资源整理，不是运行应用后重新采集
- 页面来源字段用于定位实现文件，便于从截图回溯到 Flutter 页面代码
