# 设置与维护

## 设置页里有哪些内容

当前设置页主要分成几类：

- 服务器信息
- 应用设置
- 提供商设置
- 缓存管理
- 诊断与更新

## 服务器信息

可以直接看到：

- 当前连接的线路标签
- 当前服务器地址
- 当前用户名
- 当前认证方式

这部分很适合用来确认你现在到底连的是哪条线路。

## 线路自动回退

开启后，如果你手动选择的线路挂掉，客户端可以自动切换到其他可用线路。

这适合：

- 家里一条内网地址
- 外面一条公网地址
- 平时想手动固定，但断线时又不想完全不可用

## 主题设置

支持：

- 跟随系统
- 固定浅色
- 固定深色
- 自定义主色调

## 音质设置

支持两种策略：

- 全局固定音质
- 按网络环境自动切换

如果开启按网络自动切换，可以分别设置：

- Wi-Fi 音质
- 移动数据音质

## 切歌淡入淡出

支持在 `0` 到 `3` 秒之间调整。

如果你更重视衔接感，可以适当增加；如果你更重视响应速度，可以直接关闭。

## 歌词提供商

当前客户端支持对歌词源进行启用、禁用和排序。

内置来源包括：

- 服务端歌词
- LRCLIB
- 网易云歌词
- 自定义源

排序越靠前，优先级越高。

## 封面提供商

当前支持：

- 服务端封面
- Fanart.tv
- MusicBrainz Cover Art Archive
- 自定义源

其中 Fanart.tv 支持单独配置 API Key。

## 缓存管理

缓存管理页可以查看和处理：

- 音频缓存
- 图片与资源缓存
- 歌词缓存
- 已下载歌曲

你可以在这里清理缓存，而不会影响服务器端音乐库。

## 统计信息

统计页可以查看：

- 歌曲、专辑、歌手数量
- 总播放次数
- 收藏统计
- 缓存统计
- 节省的移动网络流量估算

## 日志导出

如果要排查问题，优先使用“导出日志”。

导出的内容来自客户端内存日志缓冲区，适合：

- 自查网络切换
- 定位下载失败
- 提交 issue 时附带问题上下文

## 检查更新

设置页支持从 GitHub Releases 检查最新版本。

如果检测到新版本，页面会直接展示：

- 当前版本
- 最新版本
- 更新说明
- 可下载资产
